About This App

This web app is designed to help project managers visualize how quickly each member of their team contribute to the ongoing evolution of their repositry as well as how often.

Below is a heatmap how many pull requests each team member resolves as well as how quickly they do so. The y-axis lists the team members contributing to the repositry and the x-axis indicates how long it takes for a pull request to be resolved.

The color of each heatmap cell represents how many pull requests a given developer resolves in a given timeframe.
